# 🎯 每日習慣與獎勵追蹤 Web App

這是一個簡單的 Streamlit 應用，讓你可以每天記錄習慣並累積分數，換取個人獎勵。

## ✅ 功能

- 自訂每日習慣與得分
- 打卡後自動累積分數
- 用分數兌換自訂獎勵
- 即時呈現分數狀態

## 🚀 部署方式

1. 登入 [Streamlit Cloud](https://streamlit.io/cloud)
2. 建立一個 GitHub Repository，並上傳以下三個檔案：
   - app.py
   - requirements.txt
   - README.md
3. 在 Streamlit Cloud 新建應用，選擇 repo 並輸入 `app.py` 作為執行檔案

成功後就可以在手機上打開應用啦！

## 📦 所需套件
```
streamlit
```
